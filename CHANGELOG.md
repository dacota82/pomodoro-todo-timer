# Changelog

이 프로젝트는 [Semantic Versioning](https://semver.org/lang/ko/)을 따릅니다.

형식: `MAJOR.MINOR.PATCH`

- **MAJOR** — 큰 변경, 호환이 깨질 수 있는 개편
- **MINOR** — 새 기능 추가 (기존 사용 방식은 유지)
- **PATCH** — 버그 수정, UI 다듬기, 작은 개선

---

## [1.0.0] - 2026-07-10

### Added
- 다크 모드 플립 클럭 뽀모도로 타이머

- localStorage로 할 일·타이머·오늘 완료 수 유지
- GitHub Pages 배포

### Changed
- TO-DO 입력창·「+ 추가」 버튼 가시성 개선
- 「오늘 완료」 숫자를 완료한 할 일 개수로 반영
